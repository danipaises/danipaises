Instalador Windows

Este script configura o registro do sistema para que arquivos .dpfc abram automaticamente no seu navegador padrão e exibam o nome oficial "Dani Países File Converter".

Instrução: Após baixar, dê dois cliques no arquivo Instalar_DPFC.reg e clique em "Sim" quando o Windows perguntar.

Linux Script

Gera um script Bash automático que utiliza o comando xdg-mime para registrar o tipo MIME application/x-dpfc no seu ambiente desktop (GNOME, KDE, XFCE).

$ chmod +x instalar_dpfc_linux.sh
$ ./instalar_dpfc_linux.sh
> Registering MIME type... Done.

Configuração macOS

O macOS requer associação manual por segurança. Siga os passos:

  1. Gere seu primeiro arquivo .dpfc usando o Conversor.
  2. Clique com o botão direito no arquivo e selecione Obter Informações (Get Info).
  3. Na seção "Abrir com" (Open with), selecione Google Chrome ou Safari.
  4. Clique no botão Alterar Tudo... (Change All) para aplicar a todos os arquivos futuros.

Android & iOS

Android

Ao tocar no arquivo, o sistema pode dizer que não há aplicativo para abrir.

1. Selecione o arquivo no seu Gerenciador de Arquivos.
2. Toque em "Abrir com".
3. Escolha Chrome ou "Visualizador de HTML".

iOS (iPhone/iPad)

O iOS mostra uma prévia rápida do código ou uma tela branca.

1. Toque no ícone de Compartilhar (quadrado com seta).
2. Procure e selecione o aplicativo Chrome na lista.
3. A interface do Dani Países carregará imediatamente.
